1. Diviser un fichier:

Comme vous le voyez l'interface est très intruitive. Je vais maintenant vous décrire tous les éléments nécessaire pour votre première division.

Vous devez sélectionner avant le fichier à diviser. Pour cela cliquez sur le bouton contenant trois points ou appuyez sur CTRL+o.

Une fois le fichier à diviser sélectionné dans la boite de dialogue de sélection de fichier. La taille du fichier apparaît et le chemin des divisions est mis dans la deuxième boite à texte jaune.
Par défaut, ce chemin est le répertoire ou se trouve le fichier à diviser. Pour le changer soit cliquez sur le deuxième boite contenant les trois points soit appuyez sur CTRL+d.

Vous devez maintenant spécifier la taille de chaque division pour cela fixez la dans cette partie de la fenêtre:

Vous pouvez choisir soit "Taille prédéfinie" (quelques tailles très utilisées comme 1.38 MO ou 1.44 MO). Ou spécifier la taille manuellement.
Si vous souhaitez spécifier la taille manuellement cliquez sur le bouton radio "Autre Taille" puis spécifiez la dans la boite à texte verte.

2. Rassembler des fichiers:
Après la division du fichier c:\archive.zip. Le chemin des divisions contiens plusieurs fichiers ayant des extentions du type u001, u002, u003, etc.

Pour rassembler ces fichiers, Pour faire cela rien de plus simple!

Il faut avant tout cliquer sur la case à cocher "Rassembler des divisions" pour pour le racourçi clavier ALT+r:

Cela aura pour effet de transformer l'interface en une machine à rassemblement :-)

Cette interface verte désactive le choix de la taille d'une division puisqu'il est inutile dans ce mode ;-)

Vous devez maintenant sélectionner le fichier ayant l'extention u001 (la première division) et optionellement lerépertoire ou vont se rassembler les division.
